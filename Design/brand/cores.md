# Cores — template (editar por proposta)

Copie este ficheiro em cada repo novo e preencha. O Cursor deve aplicar em `src/index.css` (variáveis HSL).

## Produto (ex.: Factorial)

| Token | HSL | Uso |
|-------|-----|-----|
| `--primary` | `347 100% 60%` | Vermelho Factorial — botões, slides dark |
| `--secondary` | `347 50% 18%` | Fundo slides `bg: "neutral"` |
| `--accent` | `37 100% 63%` | Âmbar — destaques |
| Sidebar | `347 80% 12%` | Barra lateral do apresentador |

## Cliente (opcional)

Se a proposta misturar cor do cliente na capa apenas, descreva aqui. **Não alterar** escala tipográfica nem canvas 1920×1080.
