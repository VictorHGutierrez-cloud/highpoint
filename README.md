# High Point × Factorial HR

Interactive proposal site for **High Point Group** (South Africa).

## Live site

After GitHub Pages is enabled:  
**https://victorhgutierrez-cloud.github.io/highpoint/**

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:5173/highpoint/`

## Build

```bash
npm run build
npm run preview
```

## Structure

| Route | Page |
|-------|------|
| `/highpoint/` | Landing (cinematic hero) |
| `/highpoint/proposta` | Slide deck (20 slides) |

## Source

- Discovery transcript: `Highpoint.txt`
- SOW reference: `SOW-Factorial-HR-Implementation.md`

## Deploy

Push to `main` — GitHub Actions builds and deploys to Pages automatically.

**One-time setup in GitHub:** Settings → Pages → Source: **GitHub Actions**.
